# VExim Plugin Repository
This is just a simple repository for [VExim Web UI](https://github.com/MrSleeps/VExim-Web-UI) plugins.

It allows the vw:plugin command to make sure the list of plugins is up to date.

You will need to have installed [VExim Web UI](https://github.com/MrSleeps/VExim-Web-UI) for this to work.
